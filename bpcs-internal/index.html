Metrics

All data for all metrics comes from Google Sheets. No data is stored internally with this scorecard. The instructions below are specific to the spreadsheet used.

Adding A New Metric

Each row in a Sheet represents a single metric and includes the following fields:

  • metric: This is the name of the metric that will be displayed at the top of the metric on the scorecard.
  • tooltip: Text that will be displayed in a tooltip when mousing over the metric on the scorecard.
  • before: This is the text that is displayed immediately before the metric goal. For example, if you have a metric that should display dollar amounts, the before column should contain the symbol.
  • after: This is the text that is displayed immediately after the metric goal. For example, if you have a metric that should display a percentage, the after column should contain the % symbol.
  • status: This is a color representation of the difference between the goal value and the actual value. The accepted values are:
    • red: missed goal expectations
    • yellow: didn't quite meet the goal expectation, but within an acceptable threshold
    • green: met or exceeded goal expectations
    For example, if your goal is 100% but your actual amount is 85%, you can choose yellow, to indicate this is a warning
  • icon: The icon that best represents the metric. Not required but selected from the dropdown.
  • month-g: The goal for the indicated month. This should be a number, with no percentage or dollar signs (those are added via before and after).
  • month-a: The actual number for the indicated month. This should be a number, with no percentage or dollar signs (those are added via before and after).

Scorecards

All the data, including the scorecard list, comes from Google Sheets. The instructions below are specific to the spreadsheet used.

Adding A New Person

  1. Open the Google Sheet.
  2. Add a sheet. The name of the sheet must not contain any spaces, but hyphens are allowed.
  3. Refresh the scorecard. Your new dashboard will be listed in the Scorecards menu.

Editing An Existing Scorecard

  1. Open the Google Sheet.
  2. Rename the sheet. The name of the sheet must not contain any spaces, but hyphens are allowed.
  3. Refresh the scorecard. Your updated dashboard will be listed in the Scorecards menu.
