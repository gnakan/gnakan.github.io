Production Apps & Services

I've been the solution architect, development lead or a developer for a large number of products released in large scale production environments. I've had the pleasure to create products for a variety of companies including Blueprint Consulting Services, Microsoft, Lockheed Martin and USA Today/Gannett.

As a full-stack developer, I work heavily with MongoDB, Node.js and Angular but I'm also quite fond of Python, Flask and MySQL. I've also worked as a Microsoft .NET developer, specifically with C#, ASP.NET and SQL Server. I am an advocate for microservice architecture vs monolithic systems. I love pragmatic REST API design that indulges developers rather than dogma-driven API design that favors strict architecture adherance.
