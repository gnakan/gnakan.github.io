The Logo

Dark with color

Preferred way to use the Radar logo is over a white background.

White with color

If the logo is to be used on a dark background, text should be white.

The Icon

Dark with color

Preferred way to use the Radar icon is over a white background.

White with color

If the icon is to be used on a dark background, icon should be white.

The Typeface
